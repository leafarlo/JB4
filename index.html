⚙️ Email Setup Guide
How to add emails privately via GitHub Secrets — zero emails stored in the repo
🔐
How it works

Emails are stored as GitHub Secrets — encrypted, never visible in code, never in git history. Each resident has a secret named like EMAIL_G1_M1. The reminder script reads these at runtime via environment variables.

Go to your GitHub repo → Settings → Secrets and variables → Actions → New repository secret and add each one below.

Remove calendar events

To delete all events at once:

  1. Open your calendar app (Google Calendar, Apple Calendar, Outlook…)
  2. Find the calendar named "JB4 Duties"
  3. Delete / unsubscribe from that calendar — all events disappear instantly

If you imported into your main calendar instead, you'll need to delete the events one by one — or re-import with the file below (works in Outlook).
