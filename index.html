Welcome to Jardin Botanique 4 🌱

We're glad you're here. This guide covers everything you need to know to feel at home from day one — 23 people, one community.

🏠 Common Areas

  • Shared kitchen on each floor
  • Living room / lounge area
  • Shared bathrooms & WCs
  • Laundry room (ground floor)
  • Bike storage (basement)
  • Garden / terrace

🍳 Your Kitchen Space

  • 1 fridge rack — label with your room number
  • 1 drawer in the kitchen — yours to keep
  • Label your food clearly with your name
  • Never use others' labelled items
  • Clean up immediately after cooking

☕ Supplied by Cohabs

  • Salt Free
  • Pepper Free
  • Olive oil Free
  • Coffee (shared machine) Free
  • Toilet paper (communal WCs) Free

🗑️ Duty System

  • 5 groups rotate weekly
  • Trash: Mon/Thu — bags out the evening before; Fri — out by 9am
  • Dishwasher: empty before 10:00 AM
  • Check the Calendar tab for your schedule
  • Swaps OK — inform the ambassador

📜 Key Rules

  • Quiet hours: 11pm – 8am
  • No smoking indoors
  • Guests max 3 nights/week
  • Clean common areas after use
  • Report maintenance via Cohabs app

📞 Contacts

  • House Ambassador — ask in the group chat
  • Cohabs: cohabs.com
  • Emergency maintenance: Cohabs app
  • House WhatsApp: ask ambassador

How to add emails privately via GitHub Secrets — zero emails stored in the repo
🔐
How it works

Emails are stored as GitHub Secrets — encrypted, never visible in code, never in git history. Each resident has a secret named like EMAIL_G1_M1. The reminder script reads these at runtime via environment variables.

Go to your GitHub repo → Settings → Secrets and variables → Actions → New repository secret and add each one below.
